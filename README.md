> A React Native app which can be used to keep track of collections. Like Movies, Music, Books, etc. and keep track on your progress.

## Install

```sh
pnpm install
```

## Usage

```sh
pnpm run start
```

## Run tests

```sh
pnpm run ts:check && pnpm run lint
```
